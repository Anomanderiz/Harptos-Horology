Harptos Session Notes Template

How to use:
1) Copy this into Google Docs or any word editor if you want to.
2) Fill one or more [Event] blocks.
3) If using Google Docs or other editor, download/export as Plain Text (.txt).
4) Upload the file in the app and click "Import Notes".

Rules:
- Month can be 1-12, short name (Hammer), or full app month label.
- Real-World-Date is optional, but if present it must be YYYY-MM-DD.
- Notes starts after "Notes:" and continues until [/Event].
- You can add multiple successive events in one [Event] block by repeating "Title:" and "Notes:".
